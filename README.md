# homebrew-packages

Custom Homebrew tap for personal CLI tools.

## Usage

```bash
brew tap LeeroyDing/packages https://github.com/LeeroyDing/homebrew-packages
```

### Available Formulae

| Formula | Command | Description |
|---------|---------|-------------|
| `audiobookshelf-cli` | `abs` | CLI for managing Audiobookshelf servers |
| `seerr-cli` | `seerr` | CLI for Seerr and Overseerr media managers |

### Install

```bash
# Audiobookshelf CLI
brew install LeeroyDing/packages/audiobookshelf-cli

# Seerr CLI
brew install LeeroyDing/packages/seerr-cli
```

### Upgrade

```bash
brew upgrade LeeroyDing/packages/audiobookshelf-cli
brew upgrade LeeroyDing/packages/seerr-cli
```
